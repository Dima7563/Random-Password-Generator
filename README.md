# Random Password Generator

**Автор:** Чермошенцев Дмитрий
**Дата выполнение:** Май 2026

## Описание
Random Password Generator - Графическое приложение для генерации случайных паролей с возможностью
тонкой настройки состава символов.

## Возможности
- Выбор длины пароля от 4 до 32 (ползунок + ручной ввод).
- Выбор типов символов: цифры, буквы (латиница), специальные символы.
- Генерация с использованием стандартной библиотеки `random`.
- Сохранение истории сгенерированных паролей в JSON-файл.
- Возможность копировать пароль в буфер обмена.
- Загрузка и очистка истории.

### Тестирование:
- **Минимальная/максимальная длина** минимальная длинна пароля-4 символа, максимальная длинна пароля-32 символа.
- **Только цифры:** Снять галочки с букв и спецсимволов → пароль состоит только из цифр.
- **Все типы:** Выбрать все три чекбокса → пароль может содержать буквы, цифры и спецсимволы.
- **История:** Сгенерировать несколько паролей → они отображаются в таблице с временем.
Очистить историю → таблица пустеет, файл обновляется.
- **Загрузка истории:** Закрыть и снова открыть программу → предыдущая история загружается автоматически.

### Использование:
- Выберите длину пароля (по умолчанию 12).
- Отметьте нужные типы символов: цифры, буквы, спецсимволы.
- Нажмите «Сгенерировать пароль».
- Пароль появится в поле ввода и будет добавлен в таблицу истории.
- Нажмите «Копировать в буфер», чтобы скопировать пароль.
- История автоматически сохраняется в файл password_history.json. При следующем запуске она загрузится.
- Кнопки «Сохранить историю в JSON» (дублирующая), «Загрузить историю из JSON», «Очистить историю» позволяют управлять историей.
